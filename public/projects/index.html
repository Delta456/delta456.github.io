Box CLI Maker#

Box CLI Maker is a Highly Customized Terminal Box Maker.

Features#

  • Make a Terminal Box in 8️⃣ inbuilt different styles
  • 16 Inbuilt Colors and True Color Support 🎨
  • Custom Title Positions 📏
  • Make your Terminal Box style 📦
  • Support for ANSI, Tabbed, Multi-line and Line Wrapping boxes 📑
  • Align the text according to your needs 📐
  • Unicode, Emoji and Windows Console Support 😋

Highlights#

  • 400+ Stars on GitHub
  • Used in Kubernetes minikube
  • Featured on GoLang Weekly twice (issue#340 and issue#435)
  • Featured on GitHub Radar November 2022
  • Featured on GitHub India
  • Featured on GitHub Open Source Friday
